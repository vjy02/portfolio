## Getting Started

1. Clone the repo
   ```sh
   git clone https://github.com/vjy02/portfolio.git
   ```
2. Add in Notion API keys 
    ```
    NOTION_TOKEN=XXXXX
    DATABASE_ID=XXXXX
    ```
3. Install NPM packages and run the project
   ```sh
   npm install
   npm run dev
   ```
